1 # MCH Mapping Implementation
2 
3 This is an implementation of the [C interface](../Interface/include/MCHMappingInterface/SegmentationCInterface.h) of the MCH mapping.
4  The C function themselves are implemented using a [C++ class](src/SegmentationImpl3.h).
5  (which is not to be confused with the [C++ interface](../Interface/include/MCHMappingInterface/Segmentation.h)).
6 
7 This implementation is the first complete version of the original idea
8  to move to a self-contained mapping, i.e. mapping fully contained into
9  code, without requiring external files (either ASCII or OCDB files).
10 
11 Note that most of the files here (the `Gen*` ones) have been _generated_
12 with [alo/jsonmap/codegen](https://github.com/mrrtf/alo/tree/master/jsonmap/codegen)
13 from a [JSON](https://www.json.org) representation of the mapping.
14 
15 Internally we use a data structure meant for spatial searching : a `R-tree`
16 (from the [Boost Geometry Index](http://www.boost.org/doc/libs/1_66_0/libs/geometry/doc/html/geometry/spatial_indexes/introduction.html) library) to store and query the pads.
